# idomarketing

> A Laravel 13 + Inertia (React) marketing application.

`idomarketing` is a web application built on the Laravel React Starter Kit: a Laravel 13 backend serving an Inertia v3 single-page app with a React 19 + TypeScript frontend. It ships a complete authentication and account-settings foundation (including two-factor auth and passkeys), ready for marketing features to be built on top.

## Quick Start

```bash
# Prerequisites: PHP 8.4, Composer, Node.js + npm, and Laravel Herd
composer run setup   # install deps, create .env, generate key, migrate, build assets
composer run dev     # start the dev environment (app server + queue + logs + Vite)
```

The app is served by [Laravel Herd](https://herd.laravel.com) at **https://idomarketing.test**.

See [Getting Started](docs/getting-started.md) for the full setup walkthrough.

## Key Features

- **Full authentication** — login, registration, password reset, email verification (via Laravel Fortify).
- **Two-factor auth & passkeys** — TOTP with recovery codes and WebAuthn passkeys, both password-confirmed.
- **Account settings** — profile update/delete, password change, light/dark appearance.
- **Typed frontend routing** — Laravel Wayfinder generates TypeScript route/action helpers from PHP routes.
- **Modern frontend** — React 19 (React Compiler enabled), Tailwind v4, shadcn-style UI primitives.
- **Quality gate** — Pest tests, Pint, PHPStan/Larastan, ESLint, and Prettier wired into one CI command.

## Example

Server routes render React pages directly — no separate API:

```php
// routes/web.php
Route::inertia('/', 'welcome')->name('home');
```

```tsx
// resources/js/pages/welcome.tsx — rendered client-side by Inertia
import { dashboard } from '@/routes';           // typed helper generated by Wayfinder

<Link href={dashboard()}>Go to dashboard</Link>
```
